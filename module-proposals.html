4 Propuestas para el Módulo Showroom

Este documento presenta cuatro layouts alternativos para el módulo de presentación de productos de Nira Labs. Cada propuesta explora una estrategia de presentación diferente: desde un bento grid modular hasta una lista editorial interactiva. Todos usan contenido real y los tokens de marca del sitio.

Propuesta A

Bento Showroom

Grid asimétrico: Flowbit ocupa el ancho completo con mockup inline. Dance Yo y Tripyo en fila de dos columnas inferiores.
